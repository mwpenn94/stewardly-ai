CHART BUG ROOT CAUSE:
The WealthProjectionChart expects data in format: { year: N, [dataKey]: value }
The dashboard maps: (r.snapshots || []).map(s => ({ year: s.year, [chartMetric]: s[chartMetric] || 0 }))
But the HE result is NOT { snapshots: [...] } - it's DIRECTLY an array!
So r.snapshots is undefined, and the chart gets empty data!

FIX: Change `r.snapshots` to just `r` since HE.simulate returns the array directly.
